# Student Control System

Python command-line application developed to manage student records, grades and academic performance.

## Overview

Student Control System is a console-based application that demonstrates object-oriented programming concepts, file handling and data management in Python. The system allows users to manage student information through a simple menu-driven interface.

## Tech Stack

- Python
- CSV

## Main Features

- Add students
- View all students
- Calculate general average
- Show top three students
- Show failed students
- Delete students
- Import and export CSV files

## Project Purpose

This project was developed to strengthen Python programming skills, object-oriented design and file management while building a practical academic management application.
