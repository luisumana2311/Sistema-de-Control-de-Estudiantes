# Changelog

All notable changes to this project will be documented in this file.

The format is inspired by Keep a Changelog, and this project follows practical semantic versioning while it evolves as a portfolio project.

## [Unreleased]

### Added

- `src/student_control_system/` package structure.
- GitHub Actions workflow for automated unit tests.
- Documentation folder for screenshots and portfolio assets.
- Contribution guidelines.

### Changed

- Console entry point now loads the application from the `src/` package.
- README updated with badges, project structure, docs paths, and roadmap status.
- Console confirmation accepts `yes`, `y`, `no`, and `n`.

## [0.2.0] - P1 Functional Improvements

### Added

- Student search by full or partial name.
- Student editing flow for name, section, and grades.
- Aligned console tables for student reports.
- Menu option validation.
- Unit tests for search, updates, table output, and menu validation.

## [0.1.0] - P0 Professional Foundation

### Added

- Professional README.
- `.gitignore`.
- CSV validation and safer import behavior.
- Basic unit tests.

### Removed

- Generated Python cache files from version control.
